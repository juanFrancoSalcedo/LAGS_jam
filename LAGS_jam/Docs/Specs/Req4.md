## 4 Estamina del personaje

**Funcionalidad**
el jugador debe tener un medidor de estamina, cuando llega a cero se desmaya
hasta que se vuelva a recuperar lentamente y volver a despertar

**Programación** 
- medidor, servicio
- Freeze del movimiento del personaje
- cada acción (picar, mover carrito) debe consumir estamina

**Arte**
- 2d UI del medidor
- 2d Animacion del personaje desmayado
