# Changelog
All notable changes to this package will be documented in this file.

## [3.0.1] - 22-02-2026

# Added
- Added 'Use Scaled Time' advanced parameter.

# Fix
- Correct interpolation with Volume Weight.

## [3.0.0] - 11-02-2026

# Added
- Support for Volumes.

- Removed support for 2022.3.

# Fix
- Fixed resolution-dependent blur effect to ensure consistent visual results across different screen resolutions.

- Documentation URL fixed.
